# Solvion Website

Solvion Website

Landing page for Solvion AI built with Next.js and TailwindCSS.
Docker-ready for deployment.

## Tech Stack

- Next.js
- TailwindCSS
- TypeScript
- Docker

## Run Locally

Install dependencies

npm install

Run development server

npm run dev

Website will be available at:

http://localhost:3000

## Build Production

npm run build

## Run with Docker

Build image

docker build -t solvion-web .

Run container

docker run -p 3000:3000 solvion-web
